## 📁 Task Details

### Task 1 - To-Do List Application
A command-line based To-Do List app that allows users to:
- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as done
- ✏️ Update existing tasks
- 🗑️ Delete tasks

### Task 2 - Calculator
A simple calculator that performs basic arithmetic operations:
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- Handles division by zero errors

### Task 3 - Password Generator
A password generator app that allows users to:
- 🔢 Specify password length (6-32)
- 🔠 Include uppercase letters
- 🔢 Include numbers
- 🔣 Include symbols
- Generates strong and random passwords

### Task 4 - Rock Paper Scissors Game
A fun game against the computer with:
- 🪨 Rock, 📄 Paper, ✂️ Scissors choices
- 🎮 Game logic to determine winner
- 📊 Score tracking
- 🔄 Play multiple rounds

### Task 5 - Contact Book
A complete contact management app with:
- ➕ Add new contacts (name, phone, email, address)
- 📒 View all contacts
- 🔍 Search contacts by name or phone
- ✏️ Update contact details
- 🗑️ Delete contacts
